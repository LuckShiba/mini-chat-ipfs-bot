# Mini-chat-ipfs bot

Example bot made for [mini-chat-ipfs](https://github.com/Takasakiii/mini-chat-ipfs)

Client based in [mini-chat-ipfs](https://github.com/Takasakiii/mini-chat-ipfs).
Handler based in [ZuraaaBot](https://github.com/zuraaa-projects/ZuraaaBot)

License: GPL 3.0 (need to follow ZuraaaBot's license :$)

A HTTP IPFS Server is necessary to this works. Check [`example.env`](example.env) and copy it to `.env`.
